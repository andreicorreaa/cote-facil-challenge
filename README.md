# CoteFácil

An client developed for the Software Engineer Frontend job at CoteFácil.

This project has as main libraries Node.js, React + Vite, Typescript.

## Prerequisites

- node >= 18.13.0
- yarn >= 1.22.19

## Installation

Install all packages
```bash
yarn install
```

## Usage

```bash
yarn dev
```

## Tests

```bash
yarn test
```
